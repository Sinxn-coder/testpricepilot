Quick request example

                
Use the same key from this preview in your backend service.

              

              
curl -X POST /calculate-price \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "base_price": 10,
    "country": "IN",
    "currency": "USD",
    "min_margin": 10
  }'

Keep keys on the server and inject them through configuration.

PRICEPILOT_API_URL=https://your-api-host

Security checklist

              

              
    
                
  • Never expose production keys in browser JavaScript, public repos, or client bundles.
    • 
                
  • Route all authenticated calls through your own backend or serverless layer.
    • 
                
  • Rotate keys immediately if logs, screenshots, or test environments leak them.
    • 
                
  • Use environment variables or platform secret managers for deployment.
    •
